# Robodog Documentation

Robodog is equipped with a variety of skills and tools to enhance its functionality. Here is how you can configure and use them:

## Configuration
The skills and PARA method locations can be configured using the `skills_config.json` file. This file allows you to set the `SKILLS_ROOT` and `PARA_ROOT` directories based on your operating system. For example:
- **Windows**: `SKILLS_ROOT` might be set to `G:\\My Drive\\06_Skills`.
- **Linux**: `SKILLS_ROOT` might be set to `/home/username/06_Skills`.
- **Mac**: `SKILLS_ROOT` might be set to `/Users/username/Documents/06_Skills`.

Ensure that all skill paths use `SKILLS_ROOT`, and PARA method locations use `PARA_ROOT`.
